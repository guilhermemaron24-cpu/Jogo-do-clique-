# Sons do Jogo

Esta pasta deve conter os seguintes arquivos de áudio:

## Sons Necessários

- `start.mp3` - Som de início de jogo
- `hit.mp3` - Som ao acertar um alvo
- `miss.mp3` - Som ao errar (opcional)
- `gameover.mp3` - Som de fim de jogo
- `sector-transition.mp3` - Som de transição entre setores (futuro)
- `boss-appear.mp3` - Som de chefe aparecendo (futuro)
- `boss-defeat.mp3` - Som de derrota de chefe (futuro)

## Nota

O sistema de áudio está preparado para funcionar mesmo sem os arquivos. Se os arquivos não existirem, o jogo continuará funcionando normalmente, apenas sem sons.

Para adicionar sons:
1. Adicione os arquivos MP3 nesta pasta
2. Use nomes exatos conforme listado acima
3. Recomendado: MP3 128kbps para balancear qualidade e tamanho

## Placeholders adicionados

Os arquivos abaixo foram adicionados como placeholders silenciosos (vazios). Substitua por arquivos MP3 reais para obter efeitos audíveis:

- `boss-appear.mp3`
- `boss-hit.mp3`
- `boss-defeat.mp3`
